# Pixel 6 Home Dashboard

Static GitHub Pages dashboard plus a Google Apps Script calendar endpoint.

1. Deploy `Code.gs` in Google Apps Script as a web app.
2. Paste its `/exec` URL into `config.js`.
3. Set your coordinates and timezone in `config.js`.
4. Publish the repository with GitHub Pages.
5. Open the Pages URL on the Pixel in a kiosk/fullscreen browser.
